Byzantine Agent Teams

Single-agent outputs are unreliable. Byzantine fault tolerance fixes this.

THE PROBLEM
    One agent = one coin flip. Hallucination, drift, confident fabrication.

5 DRAFTERS
    Each writes independently. No communication. No groupthink.

3 SYNTHESIZERS
    Each reads all 5 drafts. Weaves best elements. Finds consensus.

1 ARBITER
    Reads all syntheses. Selects strongest. Refines final.

RESULT
    Richer than any single draft. ~35 agents for this project.
